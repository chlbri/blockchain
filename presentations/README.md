## 🚀 **Utilisation :**

### Pour ChatGPT PowerPoint :

1. Utiliser le fichier `ChatGPT-PowerPoint-Prompt.md` comme prompt
2. Attacher tous les fichiers `slide-X.md` de la version souhaitée
3. Préciser "version 5min" ou "version complète"

### Pour modification manuelle :

- Chaque slide est indépendante et modifiable séparément
- Les images sont organisées par slide pour faciliter la gestion
- Chemins relatifs permettent la portabilité

### Avantages de cette structure :

- ✅ **Modulaire** : Modification slide par slide
- ✅ **Organisé** : Images séparées par slide
- ✅ **Portable** : Chemins relatifs
- ✅ **Évolutif** : Ajout facile de nouvelles slides
- ✅ **Compatible** : Avec tous les outils de conversion
- ✅ **Maintenu** : Structure claire pour les mises à jour
